# LetterWiz (Modern Letter Generator, also known as Genletter)
## Tech stack used: 
#### ~ Backend: NodeJS, ExpresJS
#### ~ Frontend: ReactJS
#### ~ Database: MongoDB
#### ~ Login: Google OAuth
